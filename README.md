# Trabajo con Git

Ejercicio en equipo para practicar flujo de trabajo colaborativo con Git y GitHub: cada integrante contribuyó con una sección de una maqueta HTML de una interfaz estilo Pinterest ("Pinterest 2.0"), identificada por comentarios con su nombre en el código.

## Tech stack

- HTML5 puro (sin CSS ni JS)

## Contenido

`index.html` contiene las siguientes secciones, cada una aportada por un integrante distinto:

| Sección | Autor (según comentario) |
| --- | --- |
| Nav lateral izquierdo (iconos) | SCAR |
| Barra de búsqueda y categorías | KALEB |
| Nav lateral derecho (perfil y cuenta) | JUSEF |
| Contenido principal (feed de artículos) | TONY NAPOLES |
| Selector de preguntas frecuentes | JONATAN |
| Footer | JORG ADAN |

Es una maqueta de práctica: las imágenes del feed usan `src="#"` como placeholder y los enlaces no tienen destino real.

## Cómo verlo localmente

No requiere instalación ni build.

```bash
open index.html
```
